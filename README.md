# SquidGameUnity3D
Implementation of green light red light of squid game in unity.

## Features
✔️ Voxel art models 3D.\
✔️ Sound effects.\
✔️ Simple player controls.\
✔️ Movement detection.\
✔️ Multiple modifiable parameters from the element inspector.
